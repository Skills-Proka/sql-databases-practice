# SQL Databases Practice

Практические задания по SQL с использованием четырёх баз данных.
СУБД: **PostgreSQL** (pgAdmin 4).

## Цель проекта

Отработка навыков создания таблиц, наполнения данными и написания
SQL-запросов различной сложности: JOIN, UNION ALL, CTE, оконные функции,
рекурсивные запросы (RECURSIVE).

## Структура репозитория
```
sql-databases-practice/
├── 01_vehicles/ — Транспортные средства
├── 02_car_racing/ — Автомобильные гонки
├── 03_hotel_booking/ — Бронирование отелей
└── 04_organization/ — Структура организации
```

Каждая папка содержит:
- `create_tables.sql` — скрипт создания таблиц
- `insert_data.sql`   — скрипт наполнения тестовыми данными
- `BD*_Zad*.sql`      — решения задач

## Базы данных

### 1. Транспортные средства (`01_vehicles`)
Таблицы: `Vehicle`, `Car`, `Motorcycle`, `Bicycle`
| Задача | Описание | Баллы |
|--------|----------|-------|
| BD1_Zad1 | Мотоциклы: мощность > 150 л.с., цена < 20к, тип Sport | 3 |
| BD1_Zad2 | UNION ALL: авто + мотоциклы + велосипеды по критериям | 3 |

### 2. Автомобильные гонки (`02_car_racing`)
Таблицы: `Classes`, `Cars`, `Races`, `Results`
| Задача | Описание | Баллы |
|--------|----------|-------|
| BD2_Zad1 | Лучший автомобиль каждого класса по средней позиции | 4 |
| BD2_Zad2 | Автомобиль с наименьшей средней позицией среди всех | 4 |
| BD2_Zad3 | Классы с наименьшей средней позицией, все авто этих классов | 4 |
| BD2_Zad4 | Авто с позицией лучше средней по своему классу (мин. 2 авто в классе) | 4 |
| BD2_Zad5 | Классы с наибольшим числом авто с плохой позицией (> 3.0) | 4 |

### 3. Бронирование отелей (`03_hotel_booking`)
Таблицы: `Hotel`, `Room`, `Customer`, `Booking`
| Задача | Описание | Баллы |
|--------|----------|-------|
| BD3_Zad1 | Клиенты с > 2 бронированиями в разных отелях | 4 |
| BD3_Zad2 | INTERSECT: клиенты с > 2 бронями в разных отелях И потратившие > 500$ | 4 |
| BD3_Zad3 | Категоризация отелей и предпочтения клиентов | 4 |

### 4. Структура организации (`04_organization`)
Таблицы: `Departments`, `Roles`, `Employees`, `Projects`, `Tasks`
| Задача | Описание | Баллы |
|--------|----------|-------|
| BD4_Zad1 | RECURSIVE: все подчинённые Иванова с проектами и задачами | 4 |
| BD4_Zad2 | RECURSIVE: то же + количество задач и прямых подчинённых | 4 |
| BD4_Zad3 | RECURSIVE: менеджеры с подчинёнными, рекурсивный счётчик подчинённых | 4 |

## Запуск проекта

1. Установить [PostgreSQL](https://www.postgresql.org/) и pgAdmin 4
2. В pgAdmin создать базу данных с нужным именем
3. Открыть **Query Tool** для созданной БД
4. Запустить `create_tables.sql` → затем `insert_data.sql`
5. Запустить нужный файл задачи и проверить результат

## Технологии

- PostgreSQL 16+
- pgAdmin 4
